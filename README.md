basic example Makefile stuff


clone this repository with the command `git clone https://github.com/jarrede99/MakeExample.git`

file0.c contains the 'main' function and is dependant on all others. others are all independant.
each file has one function that takes and returns a single int and just changes its value a little
file0.c is just a call to each external function in between some print statements.
